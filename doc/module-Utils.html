(static) checkFileAndDo(filePath, interval, maxTime, criteria, timeElapsed)

Check for a file's existence and perform callback. Can be used both for waiting for a lock file to disappear, and for waiting until a certain file has been created.

Parameters:
Name Type Description
filePath String

Path to file. Required
interval Number

Interval between each file check, in ms. Optional. Default 1000.
maxTime Number

The maximum time we try until we give up, in ms. Optiona. Default 10000.
criteria String

What the success criteria is. "notexists": Wait until file in question disappears. "exists": Wait until file in question appears. Optional. Default "notexists".
timeElapsed Number

How much time has passed since we started. Used for internal, recursive calls. Optional.

(static) getDirs(basepath, depth, callback, dir)

Get all directories recursively from basePath. Stops at depth and calls callback function.

Parameters:
Name Type Description
basepath String

Path to start at. Required.
depth Number

Depth to search. Counts down for every recursive call, and when it reaches zero, the callback function is called. Required.
callback function

Callback function to call when finished. Should accept parameter containing array of absolute paths for all the directories found. Required.
dir Array

Directories found so far, and to build on. Optional.
