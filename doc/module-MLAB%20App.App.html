checkLockAndCompile(platform, timeElepased, callback)

Checks if lock file is present for app/platform. If not, starts compilation. If it is, wait for a period, and try again. This can continue until a max time limit is reached.

Parameters:
Name Type Description
platform String

Platform to compile for. Required.
timeElepased Number

Milliseconds since we first tried. Must be parameter for recursive use of this function. Optional.
callback function

Called when compile is finished, or we have met our time limit and given up. Required.
